# About this documentation

This documentation was originally maintained as a GitHub wiki at [datacharmer/dbdeployer](https://github.com/datacharmer/dbdeployer/wiki).

It has been migrated into the repository under `docs/wiki/` so it can be versioned alongside the code and kept up to date with the new ProxySQL maintainership.

Some pages may reference older MySQL versions or features. The core concepts remain the same.

For ProxySQL-specific features, see the [ProxySQL Integration Guide](../proxysql-guide.md).
